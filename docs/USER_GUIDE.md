## What is StitchX Radio?
StitchX Radio is a modern, interactive platform for real-time race intelligence and social sentiment analysis. It’s designed for demo presentations, investor showcases, and public engagement.

---

## Getting Started
1. **Open the App:** Visit the live demo link or open the deployed site.
2. **Navigation:** Use the tabs at the top to switch between the Social Feed, Demo Guide, and Investor Pitch.
3. **Mobile Users:** The interface is fully responsive—just use your mobile browser.

---

## Using the Main Features

### Live Social Feed
- View real-time posts and updates.
- Posts are analyzed for sentiment (positive/neutral/negative) using advanced AI.
- Tooltips provide extra info on sentiment scores.

### Sentiment Analysis
- Enter your own text in the provided input box.
- Click "Analyze" to see instant AI-powered sentiment results.
- All analysis is secure—your data is never shared with third parties.

### Demo Guide Tab
- Step-by-step walkthrough for demo users and investors.
- Explains each feature and how to use it.
- Great for live presentations or self-guided exploration.

### Investor Pitch Modal
- Click the "Investor Pitch" button to view the elevator pitch and business highlights.
- Modal dialog ensures focus and clarity for investor presentations.

### Feedback Submission
- Click the "Feedback" button to open the feedback form (Google Forms).
- Submit your thoughts, suggestions, or bug reports.
- Feedback is used to improve the platform.

## FAQ
**Q: Is my data secure?**  
A: Yes. All AI analysis is proxied through a secure backend. No API keys are exposed.

**Q: How do I share feedback?**  
A: Use the Feedback button to open the form.

**Q: Who can use StitchX Radio?**  
A: Anyone! It’s designed for demos, investors, and the public.
